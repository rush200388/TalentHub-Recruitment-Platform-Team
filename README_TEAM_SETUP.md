# TalentHub Recruitment Platform — Clean Team Repository

## Structure

- `backend/` — ASP.NET Core Web API, application/domain/infrastructure layers, tests, Postman, and documentation
- `frontend/` — React/Vite client
- `docs/contributions/` — individual contribution records
- `CONTRIBUTIONS.md` — team contribution summary

## First upload

The repository owner should upload this clean master repository to `main` first.
After that, each member should use the assigned branch and module package.

## Local setup

Follow the setup guides included under `backend/` and `frontend/`.

## Security

Real secrets must be stored locally with .NET User Secrets or environment variables.
Do not commit passwords, JWT keys, API keys, SMTP credentials, or cloud connection strings.
